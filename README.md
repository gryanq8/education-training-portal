# البوابة الرقمية لقطاع شؤون التعليم والتدريب

نسخة تشغيلية أولية مبنية بـ HTML + CSS + JavaScript ومربوطة مع Supabase.

## التشغيل

1. ضع بيانات Supabase في:
   `js/config.js`

2. افتح:
   `login.html`

3. سجل الدخول بالمستخدم الذي أنشأته في Supabase.

## الملفات

- `index.html`: صفحة افتتاحية.
- `login.html`: تسجيل الدخول.
- `dashboard.html`: لوحة تشغيل الإدارات.
- `css/style.css`: التنسيق.
- `js/config.js`: إعداد Supabase.
- `js/auth.js`: تسجيل الدخول.
- `js/dashboard.js`: لوحة التشغيل وإدارة المحتوى.
